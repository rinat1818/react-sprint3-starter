# Miss Notes

A Google Keep-inspired note-taking application built with React.

## 🔗 Live Demo

[View on GitHub Pages](https://rinat1818.github.io/react-sprint3-starter/)

## 🛠️ Tech Stack

- **React** - UI library
- **React Router** - Client-side routing (HashRouter)
- **JavaScript (ES6+)** - Core language
- **CSS3** - Styling with custom properties
- **LocalStorage** - Data persistence
- **Babel** - JSX transpilation
- **Font Awesome** - Icons
- **Animate.css** - Animations

## ✨ Features

### Notes App
- 📝 Create text and image notes
- 📌 Pin/unpin important notes to top
- 🎨 Customize note background colors with color picker
- 🔍 Search and filter notes
- ✏️ Edit existing notes
- 🗑️ Delete notes
- 💾 Persistent storage (notes saved to localStorage)

### General
- 🏠 Home page with app introduction
- ℹ️ About page
- 📧 Mail app (placeholder)
- 🧭 Seamless navigation between sections

## 📁 Project Structure

```
├── apps/

│       ├── pages/     # Note pages
│       └── services/  # Note services
├── assets/css/        # Stylesheets
├── cmps/              # Shared components
├── pages/             # Main pages
├── services/          # Shared services
└── lib/               # External libraries
```

## 🚀 Getting Started

1. Clone the repository
2. Open `index.html` in a browser (or use a local server)
3. Start creating notes!
